Detailed Description

Functions used to extract random data.

Provides an interface around a random data device, typically /dev/urandom, that can be used safely.

◆ RANDOM_H_BUFEN

#define RANDOM_H_BUFEN   1024

Size of the random data ring buffer.

Requests for more than this result in a direct call to read(), requests smaller than this are read from the buffer.

random_t* random_new ( )

Allocates and opens new random object.

Returns NULL on error. Uses /dev/urandom as random device.

Example
random_t *random = random_new();
assert(random != NULL);
// use random
random_free(random);
